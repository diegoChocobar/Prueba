Comandos utilizados para gestionar nuestro proyecto con GIT / GITHUB

git init; --> Se ejecuta por unica vez para comenzar a trabajar con git determinada carpeta/proyecto

git status; --> Se utiliza para ver el estado de nuestros archivos dentro de nuestro proyecto

git add . ;--> agrega al area de trabajo todos los archivos
git add <archivo>; agrega determinado archivo al area de trabajo

git commit -m "comentario de actualizacion"; pasa del area de trabajo al area local de repositorios
git push; actualiza nuestra area remetota (nube github) con lo que tenemos en el area local

git pull; actualiza nuestra area local con lo que tenemos en el area remota (nube github)

git clone <url github>; se utiliza para clonar un repositorio (descargar el proyecto en nuestra pc)
